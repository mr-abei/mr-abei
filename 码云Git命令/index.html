初次使用Git安装及使用

1、安装下载Git

下载Git （地址：https://git-scm.com/download/win）

桌面右键如下：Git GUI Here（客户端） 和GIT Batch Here(终端)

2、首次操作码云

登陆码云

通过主页右上角 「个人设置」->「安全设置」->「SSH公钥」->「添加公钥」 ，添加上面第一步生成的 public key 复制添加到当前账户中。

3、一般的提交操作步骤 ☆ ☆ ☆ ☆ ☆

（1）

git add（添加到仓库 git add . ：全部提交）

（2）

git commit -m “描述信息”（提交到仓库）

（3）

git pull （拉取最新的项目）

（4）

git push （推送自己的项目）

4、使用分支开发

I．创建分支
  1. 在码云项目上点击＂两个分支＂，进入创建新分支，例如分支名为：FirstBranch 。

在所有分支上可以看到新的分支。

  1. 此时本地是没有新分支的，所以要先在项目目录下git pull把分支拉下来:
  2. 切换分支：

查看分支：

现在在这个分支上开发

II．提交到分支上

直接：　

git add .

git commit -m ‘描述信息’

git push　

此时提交到分支上了．

5. 合并分支到主支master上

（分支开发没有问题时可以提交到主支上）

git checkout master //切换到主支

git merge origin/FirstBranch //合并FirstBranch分支

git push //推送

合并的时候，可能会进入一个linux查看文件状态里面，输入:wq 保存即可。

这样就可以了。
